# python-rock_paper_scissors_game
# 🎮 Rock Paper Scissors Game (Python)

A simple command-line Rock Paper Scissors game built using Python.
This project is part of my learning journey as a beginner, focusing on loops, conditionals, and user interaction.

---

## 🚀 Features

* Play Rock, Paper, Scissors against the computer
* Random computer choices each round
* Score tracking (User vs Computer)
* Tie detection
* Input validation
* Option to quit anytime

---

## 🛠️ Technologies Used

* Python (Beginner Level)
* Built-in `random` module

---

## 📂 How It Works

1. The user chooses to start the game
2. Each round:

   * User selects: rock, paper, or scissor
   * Computer randomly selects one option
3. The program compares both choices:

   * Win
   * Lose
   * Tie
4. Score is updated and displayed
5. The game continues until the user quits

---

## ▶️ How to Run

1. Make sure Python is installed
2. Clone or download this repository
3. Run the program:

```bash
python main.py
```

## ⚠️ Notes

* Input must be: `rock`, `paper`, or `scissor`
* Typing `q` will exit the game
* The game runs in the terminal (no GUI)
